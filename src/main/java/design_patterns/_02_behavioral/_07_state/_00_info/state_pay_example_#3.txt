Приведена реализация шаблона State для проверки процесса оплаты заказа и выставления штрафных санкций
в случае задержки оплаты.
Данную реализацию следует в качестве упражнения попробовать переделать, заменив поле IPayState класса Order
на поле типа boolean и оценить, как это отразится на организации класса и способах вычисления сумм к оплате.
