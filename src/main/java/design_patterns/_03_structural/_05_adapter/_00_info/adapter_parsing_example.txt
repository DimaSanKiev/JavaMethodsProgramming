Пусть в систему JSON-парсинга необходимо подключить разбор XML-документа. Для этого выделяется интерфейс Parser
с методом parse(). Существующий класс JsonParser теперь должен имплементировать новый интерфейс. Для адаптации
интерфейса класса XmlParser создается класс XmlParserAdapter, объявляющий адаптируемый класс в качестве поля,
а также в качестве полей объявляет необходимые для согласования интерфейса параметры. Метод parse() адаптера
теперь может корректно вызвать метод парсинга адаптируемого класса.
